[vi 편집기 사용법]

[주요동작]
편집시작: vi hello.txt
편집종료: 이동모드에서 콜론(:)
	- w: 저장
	- wq: 저장 후 편집기 종료
	- x: 저장 후 편집기 종료

[이동모드]
    - 화살표 이동
	- h : 한 글자 왼쪽 이동
	- l : 한 글자 오른쪽 이동
	- j : 한 라인 아래로 이동
	- k : 한 라인 위로 이동
	- 0 : 현 라인의 처음으로 이동
	- $ : 현 라인의 마지막으로 이동
	- b : 현 라인에서 이전으로 단어씩 이동
	- gg, 1G : 첫 번째 행으로 이동
	- G : 마지막 행으로 이동
	- yy, Y: 현재 라인을 복사하기 위해 보관
	- p : 현재 라인 다음에 보관된 내용을 복사
	- P : 현재 라인 이전에 보관된 내용을 복사
	- . : 바로 직전 수행한 행위를 반복
	- dd: 현재 라인을 삭제
	- D : 현재 라인의 위치에서 라인의 끝까지 삭제
	
[편집모드] 이동모드에서 편집모드로 이동 및 명령
	- i : 현재 위치에 입력 시작
	- a : 현재 위치 뒤에 입력 시작
	- o : 현재 위치 다음 라인에 입력 시작
	- O : 현재 위치 윗 라인에 입력 시작
	- s : 현재 위치에 글자를 지우면서 입력 시작
	- I : 현재 라인의 맨 앞에서 입력 시작
	- A : 현재 라인의 맨 끝에서 입력 시작
	- C : 현재 위치에서부터 끝까지 지우면서 입력 시작
	- S : 현재 라인을 모두 지우면서 처음부터 입력 시작
	- cw: 현재 위치에 단어를 지우면서 입력 시작

[명령모드] 이동모드에서 명령모드로 들어갈 때 쓰는 명령
	- 콜론(:) 후 아래 명령
	- w   : 저장
	- wq  : 저장 후 종료
	- x   : 저장 후 종료
	- wq! : 저장 후 종료(강제)
	- 슬래시(/)나 물음표(?) 후 아래 명령: 검색
	- /검색어 : 현재 위치 이후부터 검색
	- ?검색어 : 현위 위치 이전으로 검색
	- n : 다음검색(next)
